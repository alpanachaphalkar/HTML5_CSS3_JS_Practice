Overflow Property in CSS

Again, these properties are useful to create layouts that can be resized depending upon the size of the user’s browser window. However, you can see an interesting phenomenon in Figure 7-34: If the content of the box takes up more space than the box is allowed because of these rules, the content can overflow out of the box. (You learn how to deal with this in the next section).As you just saw in Figure 7-34, when you control the size of a box, the content you want to fit in the box might require more space than you have allowed for it. This happens not only with the min-height and max-height or min-width and max-width properties but also for a number of other reasons.
